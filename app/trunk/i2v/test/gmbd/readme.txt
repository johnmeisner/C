Global Misbehavior Detection (GMBD) Tools
=============================================

The tools in this directory can be used for GMBD testing.
There are tools that are designed for WSU use, and tools
that are for a host PC.  See the contents of each directory
for the user guides or instructions on how to operate each.

The following directories are included (update this file
when adding new tools):

    * scms-simulator - For receiving MBRs
